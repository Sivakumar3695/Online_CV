MiniSat for D(n,p,q) random AF instance
A research project developed to solve the D(n,p,q) random model introduced in a research paper. To be precise, this project is an extension of the research work mentioned in the original paper where further works were called for the introduced random model to be solved by modern solvers. The project is a small extension of the research work since the MiniSat used in the project is not a modern solver but its features are widely adopted in most of the modern SAT-based AF solvers

In addition to this, a new branching technique designed particularly for solving AF instance is introduced. A graphical output will be provided for the users to make easy comparison. More details are provided in the github repository.

Erdős–Rényi Graph Generator - G(n,p) model
A basic project to generate ER graph using G(n, p) random model. An output xlsx file containing Actual Vs Expected number of nodes and Actual Vs Expected node degree will be provided.
