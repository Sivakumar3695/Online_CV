  • Branching Heuristics impact analyser is a research project developed to study the impact of various branching heuristics in an Argumentation Framework solver.

  • Random CSP Generator - A simple Java project to generate random Constraint Satisfaction Problems based on one of the four random CSP Models: Model B, Model D, Model RB and Model RD.
  • Custom ER Graph Generator - A Python project to generate ER graphs using the G(n, p) random model. The project will output an XLSX file containing ‘Actual Vs Expected number of nodes’ and ‘Actual Vs Expected node degree’.
