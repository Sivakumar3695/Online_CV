Custom Handwritten Digit Predictor
The project started with a view to training a CNN model for the MNIST dataset but later extended to support multiple model training using a simplified custom framework. The following are some of the highlights of the project:
  • Train and save a CNN model.
  • Use a pre-trained model to predict one or more custom handwritten images of digits.
  • Option to perform test data evaluation on any pre-trained model.
  • Use feature visualization to understand what the system is expecting and why some of the predictions are incorrect.
  • Maximum accuracy achieved: 99.20%
  • Maximum precision attained: 96.01%

To know about the usage of feature visualization in this project, please check the Github repository.

CNN Model Trainer + Comparator
CNN model trainer + comparator is a Python project for the CIFAR-10 dataset using the PyTorch library. Better utilization of this project is to provide two models and choose the efficient model by making an obvious comparison based on the convergence speed, test accuracy and mean loss after each iteration. The project supports the usage of GPU with the help of the PyTorch library. While comparing two models, the project will provide a graph to compare the efficiency of the two models in the training process. As model training is a time-consuming process, one should prefer to use GPU for the model comparison.
Maximum accuracy achieved after various model training (using a random model): 83.74%

To know more about the project, please check the Github repository.

A study on the impact of Branching Heuristics in AF solvers
This project is a part of a research work to emphasize the importance of branching heuristics in SAT-based Argumentation Framework (AF) solvers to solve instances from the D(n,p,q) random model. Various branching techniques are used in this project to computationally establish their impact on the solvers for solving AFs generated with different values of attack probability. This research study involved graphs, random graph generation, SAT solvers & related CNF encodings, a variety of branching heuristics and essential algorithmic changes to incorporate the branching heuristics in the used SAT solvers.

Although the MiniSat is not the start-of-the-art SAT solver, this project uses it to solve random AF instances because of its extensive documentation. Also, most modern solvers have the MiniSat's core features utilized in them. Hence, it will not be challenging to study this work on modern solvers.

A new branching technique named Attack Relation Occurrence Factor Summation (AROFS) is introduced in this research work. One can run this Python code to compare the models equipped with the following branching techniques:
  • VSIDS (default)
  • BH3-BRmcha
  • AROFS

More details are provided in the Github repository.
