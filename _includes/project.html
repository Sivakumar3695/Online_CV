Custom Handwritten Digit Predictor
A project started with an objective to train a CNN model for MNIST dataset but later extended to support multiple model training using a simplified custom framework. The following are some of the highlights of the project:
  • Train and save a CNN model.
  • Use a pre-trained model to predict one or more custom handwritten images of digits.
  • Option to perform test data evaluation on any pre-trained model.
  • Use feature visualization to understand what the system is expecting and why some of the predictions are incorrect.
  • Maximum accuracy obtained: 99.20%
  • Maximum Precision obtained: 96.01%

Please check the github repository to know about the usage of feature visualization in this project.

CNN Model Trainer+Comparator
A CNN model trainer for CIFAR-10 dataset using the PyTorch library. A better utilization of this project is to provide two models and choose the better among the two by making obvious comparison based of the convergence speed, test accuracy and mean loss after each iteration. With the help of PyTorch library, GPU usage is supported. While doing model comparison, it is wise to use GPU to fasten the model training process.
Maximum Accuracy obtained after various model training (using random model): 83.74%
To know more about the project, please check the github repository.

MiniSat for D(n,p,q) random AF instance
A research project developed to solve the D(n,p,q) random model introduced in a research paper. To be precise, this project is an extension of the research work mentioned in the original paper where further works were called for the introduced random model to be solved by modern solvers. The project is a small extension of the research work since the MiniSat used in the project is not a modern solver but its features are widely adopted in most of the modern SAT-based AF solvers

In addition to this, a new branching technique designed particularly for solving AF instance is introduced. A graphical output will be provided for the users to make easy comparison. More details are provided in the github repository.
